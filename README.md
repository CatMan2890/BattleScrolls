# BattleScrolls
Minecraft plugin + resource pack

Плагин для Purpur 1.21.11 (совместим с Paper/Spigot 1.21.x) + ресурспак

# Команды и права

/scroll give <игрок> <fire|ice|chaos> [количество]` — выдать свиток. Право: `battlescrolls.give` (по умолчанию только у оперов)

Право на использование свитков: `battlescrolls.use` (по умолчанию есть у всех)

# Логика свитков
**Огонь** — урон по всем `LivingEntity` в радиусе, поджигает их на заданное число тиков, частицы пламени
**Лёд** — урон + отталкивание вверх-в сторону + сильный Slowness/Weakness/Blindness на время «нокдауна» — это и есть имитация «сбит с ног и лежит»: в ванильном Bukkit API нет официального способа насильно уложить сущность в позу лёжа, поэтому это сделано через связку эффектов и отбрасывания, что даёт тот же игровой результат (враг обездвижен и не может ничего сделать)
**Хаос** — во всех блоках сферы радиусом 10 (кроме воздуха, жидкостей и бедрока) временно подменяется материал на случайный из списка в конфиге (блоки других биомов: песок, лёд, джунглевые доски, незерак и т.д.), через настроенное время всё возвращается на исходные блоки. Это ближе к реальному изменению текстур, чем клиентские трюки — по сути глобальная подмена текстуры одного блока для всех выглядела бы читерски и без причины задела бы весь мир, а так эффект локальный, честно завязан на реальные блоки и их ванильные текстуры «других биомов»

Все числа (урон, радиус, cooldown, длительности, список блоков хаоса) — в `config.yml`
